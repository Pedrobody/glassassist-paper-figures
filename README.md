# GlassAssist — Paper figures & analysis

Figures and reproducible analysis scripts for the GlassAssist thermal-throttling
diagnosis and perception-offloading study.

**CAVA Lab, Aoyama Gakuin University.**

- `cava.py` — lab visual identity (matplotlib style)
- `bench_ocr.py` / `bench_ocr.csv` — reproducible OCR latency benchmark (GPU vs CPU)
- `figs_ocr.py` / `figs_thermal.py` — figure generators
- `fig_*.png` — generated figures
